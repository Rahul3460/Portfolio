# 🚀 Rahul Mahata - Professional Portfolio

A modern, responsive, and feature-rich portfolio website showcasing my skills, projects, and experience as a Full Stack Developer and MCA student.

## ✨ Features

- 🎨 **Modern UI/UX** - Sleek glassmorphism design with gradient accents
- 🌙 **Dark Theme** - Eye-friendly dark mode with optional light theme toggle
- 📱 **Fully Responsive** - Optimized for all devices (mobile, tablet, desktop)
- ⚡ **Fast Performance** - Optimized loading and smooth animations
- 🎭 **Interactive Elements** - Custom cursor, particle effects, typing animation
- 📊 **Projects Showcase** - Filterable project gallery with detailed information
- 📧 **Contact Form** - Functional contact form with validation
- 🎓 **Education Timeline** - Visual representation of educational journey
- 💼 **Skills Display** - Animated progress bars showing skill proficiency
- 🔍 **SEO Optimized** - Meta tags and semantic HTML for better discoverability

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid, Animations
- **JavaScript (ES6+)** - Modern vanilla JavaScript
- **Font Awesome** - Icon library
- **Google Fonts** - Poppins & Orbitron fonts
- **AOS Library** - Scroll animations
